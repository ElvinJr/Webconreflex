# 🚀 Reimagina una Landing Page con Reflex

## 🎯 Objetivo
Poner en práctica los conocimientos de desarrollo frontend utilizando **Reflex** para recrear el diseño de una landing page profesional, aplicando principios de diseño web, estructura semántica y buenas prácticas de codificación.

## 📝 Asignación
La tarea consistía en:
- Explorar plataformas de inspiración como **Dribbble**, **Behance** y **Landingfolio**.
- Seleccionar una landing page que nos inspirara y reinterpretarla.
- Planificar la estructura (header, secciones, llamado a la acción, formulario, etc.).
- Identificar los componentes necesarios (barra de navegación, tarjetas, botones, footer).
- Desarrollar la landing page con **Reflex**, usando containers, stacks, flexboxes y grids.
- Mantener una organización clara y semántica en el código.
- (Opcional) Publicar el proyecto en GitHub y desplegarlo con Reflex hosting.

## ⚙️ Lo que hicimos
- Seleccionamos el diseño de referencia en **Landingfolio**.  
- Reemplazamos imágenes decorativas (`yellow_donut`, `silver_flower`, `purple_blob`).  
- Implementamos la sección **Follow us** con enlaces funcionales:  
  - 📷 Instagram → [Perfil ElvinJr](https://www.instagram.com/elvinjr_?igsh=MWFtaWE1amN3bzEzcw==)  
  - 💡 Landingfolio → [Inspiración Inflatable](https://www.landingfolio.com/inspiration/post/inflatable?category=landing-page)  
  - 📌 Pinterest → [Wannathis](https://br.pinterest.com/wannathisone/)  
  - 🐦 Twitter/X → [Wannathis3D](https://x.com/wannathis3d)  
- Organizamos la estructura con Reflex (`navbar`, `hero`, `features`, `footer`).  
- Documentamos el proceso y conclusiones en este README.

## 🌟 Resultado
Una landing page funcional y organizada, que replica un diseño profesional con Reflex y demuestra el uso de buenas prácticas de frontend.
